# MapStack - Landing Page

Proyecto académico desarrollado como parte del curso de Desarrollo Web.

## Descripción

MapStack es una landing page diseñada para promocionar una plataforma ficticia que permite integrar mapas interactivos en aplicaciones web de forma rápida, segura y profesional.

La página incluye navegación fija, filtros dinámicos, modo claro/oscuro, contador de características visibles y un diseño responsivo básico.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Flexbox
- Variables CSS
- DOM

---

## Funcionalidades implementadas

### 1. Menú de navegación fijo
Permite desplazarse suavemente entre las secciones de la página.

### 2. Sección HERO
Incluye:
- Nombre del producto
- Eslogan
- Imagen representativa
- Botón que hace scroll automático a la sección de precios

### 3. Filtro dinámico de características
Permite mostrar características por categoría:
- Core
- Integración
- Seguridad
- Todas

También incluye un contador que indica cuántas características están visibles.

### 4. Modo claro / oscuro
Botón que permite cambiar el tema visual de la página usando clases en CSS.

### 5. Sección "Cómo funciona"
Muestra los pasos para usar la plataforma con interacción en cada paso.

### 6. Sección de planes
Incluye tres planes:
- Gratis
- Pro
- Empresarial

Además, se puede activar el modo anual mediante un checkbox.

### 7. Testimonios dinámicos
Se pueden cambiar usando botones "Anterior" y "Siguiente".

### 8. Formulario de contacto
Incluye:
- Nombre
- Correo
- Asunto
- Mensaje

Con validación básica en JavaScript.
